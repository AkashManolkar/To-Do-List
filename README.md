# To-Do List Web Application

## 📋 Description

This is a simple and interactive To-Do List web application built using **HTML**, **CSS**, and **JavaScript**. It allows users to:
- Add new tasks
- Edit existing tasks
- Delete tasks

## 🌐 Live Preview

You can run the application locally by opening the `index.html` file in any modern browser.

## 🛠️ Technologies Used

- **HTML** – for structuring the web page
- **CSS** – for styling the application
- **JavaScript** – for adding dynamic behavior and task management

## 📁 Project Structure

to-do-list/
│
├── index.html # Main HTML file
├── styles2.css # CSS file for styling
├── script.js # JavaScript file for functionality
└── README.md # Project documentation


## 🚀 How to Run

1. **Clone the repository or download the project files.**
2. Open `index.html` in any web browser.
3. Start adding your tasks!

## 📌 Features

- **Add Task** – Type a task and click "Add Task" to include it in your list.
- **Edit Task** – Click "Edit" to change the task description.
- **Delete Task** – Remove a task by clicking the "Delete" button.
